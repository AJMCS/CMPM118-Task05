# CMPM118 Task 4 - Prolog Knowledge Base

A Studio Ghibli knowledge base built in Prolog and queried 
via Python using pyswip. The KB contains facts about Ghibli films including 
genre, director, composer, and awards, with rules for querying and 
discovering relationships between films.

## Setup
1. Install SWI-Prolog: `brew install swi-prolog`
2. Create a virtual environment: `python3 -m venv venv`
3. Activate it: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run: `python3 ghibli.py`
